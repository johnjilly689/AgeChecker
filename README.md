# AgeChecker
A responsive full-stack age checker app built with HTML, CSS, JavaScript, Axios, Node.js, and Express Backened API.

# 🎂 Age Checker App
> A simple full-stack app where users enter their age,
> the frontend communicates with an Express backend via 
> Axios, and instantly displays the result.

----------------------------------------------------

## 🔗 Live Demo
[👉 Click Here To Try It](https://agechecker-4m1l.onrender.com)


--------------------------------

## ✨ Features
- ⚡ Responsive user interface
- 🔄 Frontend & backend communication via Axios
- ✅ Age validation API
- 🌐 CORS-enabled Express server

----------------------------------------------------

## 🛠️ Tech Stack
| Frontend  | Backend    |
|---------- |---------   |
| HTML5     | Node.js    |
| CSS3      | Express.js |
|JavaScript | Axios      |

---

## 🚀 Getting Started

### Installation
git clone https://github.com/johnjilly689/AgeChecker.git
cd age-checker-app
npm install

### Run The App
node server.js

Then open index.html in your browser.
